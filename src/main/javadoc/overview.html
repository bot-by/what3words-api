Unofficial Java wrapper for what3words.com API
to convert 3 word addresses into coordinates and vice versa

Please add dependency to your project: 

<dependency>
  <groupId>uk.bot-by.3wa</groupId>
  <artifactId>what3words-api</artifactId>
  <version><!-- search on Maven Central --></version>
</dependency>
Find last version on Maven Central.
